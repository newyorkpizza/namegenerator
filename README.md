namegenerator
=============

JavaScript function for generating satirical company names by combining tech terms with culinary terms.

Some examples:

	UtilityPod
    SmokedLoad
    AsynchronousLemon
    FibersDefault
    BrowserEmulation
    ServerPing
    KeySalad
    PhoneEngine
    CircuitDeprecated
    CoolFrame
    WokTweak
    ComputeMadeleine

Demo
====
See the name generator in action [right here](https://newyorkpizza.github.io/namegenerator).
