namegenerator
=============

JavaScript function for generating satirical company names by combinaing tech terms with culinary terms.

Some example names:

	UtilityPod
    SmokedLoad
    AsynchronousLemon
    FibersDefault
    BrowserEmulation
    ServerPing
    KeySalad
    PhoneEngine
    CircuitDeprecated
    CoolFrame
    WokTweak
    ComputeMadeleine
